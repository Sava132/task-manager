# Task Manager App 📝

A clean and functional Task Manager built with **Vanilla JavaScript**. This project focuses on DOM manipulation and efficient state management without external frameworks.

## 🚀 Live Demo
[View Project Live](https://sava132.github.io/task-manager/)

## ✨ Features
* **Add Tasks:** Quickly add new tasks to your list.
* **Remove Tasks:** Delete completed or unnecessary tasks with one click.
* **Interactive UI:** Smooth transitions and a modern dark theme.
* **Responsive Design:** Optimized for both desktop and mobile screens.

## 🛠️ Tech Stack
* **HTML5** - Semantic structure
* **CSS3** - Custom styling with Flexbox and Glassmorphism effects
* **JavaScript** - Core logic and DOM manipulation

## 📖 How to run locally
1. Clone the repository: `git clone https://github.com/Sava132/task-manager.git`
2. Open `index.html` in your favorite browser.
